# RA4 - AEE: Creación de un Servicio de Autenticación con Sesiones en PHP

## ¿Qué es?
Es un proyecto encargado que nos sirve para aprender cómo comprobar que el usuario se haya logueado correctamente usando PHP como comprobador, e impedir que alguien que no está logueado se pueda meter en nuestra página de bienvenida.

## Tecnologías usadas:
Los lenguajes usados: HTML y PHP.

## ¿Cómo lo puedo instalar?
Lo puedes usar con un IDE como Visual Studio Code y con un servidor como XAMPP.
